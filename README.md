# Credit_Card Transactions and Customer Analysis
Interactive Power BI dashboard for analyzing credit card transaction 
## Tools Used
- Power BI Desktop
- Power Query
## Analysis Performed 
- Revenue analysis by quarter
- Transcation analysis by card category
- Customer segmentation by income, age and profession
- Spending behavior analysis
- Interest earned and transaction amount analysis
## Dashboard Included
- Credit card Transaction Dashboard
- Credit Card customer Dashboard
## Key Insights
- Blue card category generates the highest revenue
- White-collar and self employed customer are major contributors
- Graduates from the largest revenue generating eduacation group
- Spending pattens vary across income and age groups
## Conclusion
This project demonstrates strong business intelligence and data visualization skills. It shows the ability to convert raw data into meaningful dashboards.
